## Features

### Table Columns
1. **å•é¡Œ** (Question): The question text
2. **æ­£è§£** (Correct Answer): The correct answer text
3. **ã‚ãªãŸã®å›ç­”** (Your Answer): Visual indicator and incorrect answer if applicable
   - â­• for correct answers
   - âœ– for incorrect answers
   - For incorrect answers, shows the user's selected choice in parentheses (dimmed)

### Data Flow
1. Answers are stored in `sessionStorage` as they are selected in `QuizQuestionScreen`
2. Each answer includes:
   - `questionId`: The question being answered
   - `selectedChoiceId`: The choice the user selected
   - `isCorrect`: Whether the answer was correct
   - `answeredAt`: Timestamp
3. `QuizResultScreen` retrieves these answers and maps them to quiz questions
4. The table displays the comparison between correct answers and user answers

## Benefits
- Provides clear feedback on quiz performance
- Shows exactly which questions were answered correctly or incorrectly
- Helps users learn from their mistakes by showing the correct answer
- Matches the UX requirements specified in the issue
